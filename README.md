Bakery Order Management System:
An intelligent Bakery Order Management System designed to optimize order scheduling and resource utilization using Greedy Algorithm and Dynamic Programming.
The system improves efficiency by prioritizing orders based on urgency, profit, and complexity while minimizing ingredient wastage and ensuring timely delivery.

Features:
Smart order prioritization using Greedy Algorithm
Ingredient optimization using Dynamic Programming (0/1 Knapsack)
Interactive dashboard for order tracking and analytics
Detailed order management with customization options
Inventory tracking and resource allocation

Tech Stack:
Frontend: HTML, CSS, JavaScript
Version Control: GitHub

How It Works:
Customers place orders through the system
Orders are assigned a priority score based on:
Deadline (urgency)
Order value (profit)
Production complexity

The Greedy Algorithm sorts orders for execution
Dynamic Programming optimizes ingredient usage
Orders are scheduled and processed efficiently
